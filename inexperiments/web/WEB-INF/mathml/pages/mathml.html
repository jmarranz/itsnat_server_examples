MathML Demo served with text/html MIME

This example is copied and slightly modified from this example, a MathML testing demo for WebKit.

It shows how ItsNat can automatically rebuild on load time non-HTML/XHTML namespaces in HTML/XHTML pages served with HTML MIME (text/html) for W3C browsers and remain compatible with MSIE v6-v8. ItsNat does not know MathML elements, any non-XHTML namespace is supported by ItsNat.

This example works in FireFox (FireFox supports MathML since some years ago) and very soon in WebKit based browsers (Chrome, Safari...), these browsers recognize namespaces in markup. It also works in MSIE v6-v8 ignoring namespaces (no namespace rebuilt).
